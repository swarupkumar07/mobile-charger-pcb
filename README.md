# Mobile Charger PCB

Custom PCB design for a mobile charger providing stable and efficient power output.

## Description

This project is a mobile charger PCB designed to convert input power into a stable DC output for charging mobile devices. The design is compact and optimized for efficient performance.

## Features

* Stable output voltage
* Compact PCB layout
* Efficient power conversion
* Reliable design

## Files Included

* Schematic file
* PCB layout file
* Gerber files
* Bill of Materials (BOM)

## Tools Used

* KiCad / Eagle
